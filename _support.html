Commercial curl support is offered via wolfSSL.

Daniel Stenberg, the founder and lead developer of curl, works for and with wolfSSL to provide world class support to customers.

We offer full commercial support on curl done by the masters of curl. We can handle

  • curl customization
  • curl ports to new operating systems
  • feature development
  • patch maintenance
  • bug fixing
  • backporting of fixes to older curl versions
  • upstreaming your fixes into the curl git repository
  • optimizing curl for your use case or wherever it needs to improve
  • training your developers to become better users of curl and libcurl
  • code reviews of your libcurl API use
  • security scanning your curl use
  • Cyber Resiliency Act (CRA) compliance assistance
...and more - with several different support options from basic up to full 24/7 support.
