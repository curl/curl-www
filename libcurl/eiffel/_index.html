Eiffel bindings to libcurl.

Download
Please download it with SVN from https://svn.origo.ethz.ch/eiffelstudio/trunk/Src/library/cURL

Install
Before using cURL Eiffel binding, you need to compile cURL C library in cURL's Clib folder. The command is:

finish_freezing -library
