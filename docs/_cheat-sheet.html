Tutorial

GET requests

GET a website

curl https://example.com

Follow redirects

curl -L https://example.com

Hide the progress bar

curl -s https://example.com

Output to a file

curl -o out.html https://example.com/

Set or view headers

Show response headers

curl -i https://example.com

Make a HEAD request

curl -I https://example.com

Set a header

curl https://pages.github.io -H "Host: raytracing.github.io"

POST requests

POST some JSON

curl -v httpbin.io/post --json '{"key":"value"}'

POST JSON from a file

curl -v httpbin.io/post --json @file.json

POST application/url-encoded data

curl -v httpbin.io/post --data-urlencode name=maya --data-urlencode date=2024-01-01

More

Make a DELETE request

curl -X DELETE example.com

Override DNS resolution

curl example.com --resolve example.com:443:23.220.75.245

Do basic authentication

curl -u user:pass example.com
